# X Clone 

A full-stack **X (Twitter) clone** built with the **MERN stack**, featuring authentication, post creation, image uploads, follow system, notifications, and a modern responsive UI.

---

## 🚀 Features

- 🔐 JWT-based authentication with secure cookies
- 👤 User profiles with image upload
- 📝 Create, like, comment, and delete posts
- ➕ Follow / unfollow users
- 🔔 Real-time style notifications
- 📱 Fully responsive social media UI

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, DaisyUI, TanStack Query, React Router DOM  
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, Cloudinary
